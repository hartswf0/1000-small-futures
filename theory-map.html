◎ LEGOS Theoretical Correspondences

Mapping how 1000 Small Futures / LEGOS instantiates five theoretical frameworks: Metabolarum (semiotic metabolism), Media Thermodynamics, MYTHOSLOG (formal ontology), Myth-OS (cultural computation), and LLMA (cultural production engine).

Core Thesis

LEGOS is an implementation of executable theory—it operationalizes abstract frameworks into functioning computational infrastructure. The grid, ring buffer, tetrad, and scene graph are not just tools but living instantiations of metabolic, thermodynamic, and ontological principles.

I. LEGOS ↔ Metabolarum (The Semiotic Life)

Framework Summary

Metabolarum treats media as a living organism with metabolism: ingestion, digestion, circulation, assimilation, excretion. Central problem is semiotic rot— the entropy of links, context, and meaning over time.

LEGOS Component Metabolarum Function Correspondence
Ring Buffer Digestive Tract Circular overwrite = metabolic homeostasis. Graceful degradation = healthy elimination.
Temporal Index Metabolic Pathways Tracks how information moves through system over time. Query = biopsy of metabolic state.
URL/Link Checking Rot Detection 404s = semiotic necrosis. Broken references = entropy markers in knowledge graph.
LLM Remediation Enzymatic Repair Multimodal AI = enzyme kit. OCR, TTS, summarization = metabolic rescue functions.
Scene Graph Semiotic Structure Knowledge graph that can decay. Entities/relations = structural integrity of meaning.
Export/Import Excretion/Ingestion JSON export = metabolic output. Session import = feeding new material to system.
Key Insight: The ring buffer doesn't just store—it digests. Compaction = catabolism. Old entries lose detail but preserve semantic essence. This is computational orality as metabolism.

IX. Research Questions

Critical Open Problems

1. Can semiotic rot be measured?
Design experiment: Generate 1000 scenes. Track link decay, context drift, meaning erosion over 6 months. Measure: Does LEGOS remediation (tetrad, perspective) slow rot?

2. Is the tetrad thermodynamically optimal?
Why 4 directions, not 3 or 5? Test hypothesis: Four laws = minimal complete basis for conceptual work extraction. Compare energy efficiency of tetrad vs. other frameworks.

3. What is the carrying capacity of the grid?
9×9 = 81 cells. Is this optimal for human cognition? Test: Vary grid size (5×5, 13×13, 20×20). Measure: Cognitive load, insight generation, navigation ease.

4. Can the ring buffer become self-aware?
Current: Manual compaction triggers. Future: System learns what to remember based on future query patterns. This is autopoietic metabolism—system regulating its own digestion.

5. Is LEGOS a living system?
Metabolarum claims media is alive. LEGOS has: intake (prompts), metabolism (LLM processing), circulation (ring buffer), homeostasis (graceful degradation), reproduction (forking). Is this metaphor or literal?
